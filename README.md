# County Appliance Repair

Static website for County Appliance Repair in San Mateo, CA.

## Local preview

```bash
npm run dev
```

Then open `http://localhost:4173`.

## Deploy

This project is ready for Vercel static hosting. The canonical URL is currently set to:

```text
https://county-appliance-repair.vercel.app/
```
